Infinite Integer Calculator

Adding, subtracting or multiplying two infinite integers using school mathematics.

Implement

This is my second approach to do the math. The main idea is to add, subtract or multiply using school mathematics.

Instead of using BigInt type, I use string for doing the math. In the past, no maximum string length was specified. ECMAScript 2016 (ed. 7) established a maximum string length of 253 - 1 elements. Basically, if there is any input or result that overflow the maximum length of string, the calculator probably can't handle it. In other words, it is truly the maximum bound of the calculator.

In conclusion, this calculating method is probably better than the BigInt approach. We are only limited by the maximum bound of the data types storing our inputs. Although it is not the infinite calculator as we expected, if you wish to enter a number having more than 253 - 1 digits, give it a try and let find out the result. Note that it may take some time to calculate for very large input.
